## Step-by-Step Submission

### 1. Upload APK/AAB to Release Track

**Location:** Release → Closed testing → Create release

1. Go to Google Play Console dashboard
2. Click on your Masidy app
3. Navigate to **Release** (left menu)
4. Select **Closed testing**
5. Click **Create release**
6. Upload your APK/AAB file
7. Add release notes: "Initial release for testing"
8. Click **Save**

### 2. Complete Store Listing

**Location:** Grow → Store presence → Store listing

#### 2a. App Title & Description
1. **App name:** Masidy (30 chars max)
2. **Short description:** Copy from STORE_LISTING.md (80 chars)
3. **Full description:** Paste from STORE_LISTING.md (4000 chars max)

#### 2b. App Icon & Screenshots
1. Upload app icon (512×512 PNG)
2. Click **Screenshots** tab
3. Upload 5-6 screenshots (follow SCREENSHOT_GUIDELINES.md)
4. Add captions to each screenshot
5. Drag to arrange order

#### 2c. Feature Graphic
1. Recommended: 1024×500 PNG
2. Shows at top of store listing
3. Highlights main feature

#### 2d. Category & Content Rating
1. **Category:** Productivity
2. **Content rating:** Click "Complete questionnaire"
   - Quick 5-10 minute form
   - Select age rating (3+)
   - Submit (instant approval)

### 3. Add Privacy Policy

**Location:** Policies → App policies

1. Go to **Policies** (left menu)
2. Click **App policies**
3. Scroll to **Privacy policy**
4. Paste privacy policy URL: `https://www.masidy.com/privacy-policy`
5. Click **Save**

### 4. Set Pricing & Distribution

**Location:** Monetization → Pricing & distribution

1. **Pricing:**
   - Select **Free**
   - Confirm: "Make this app always free"

2. **Distribution:**
   - Select all countries (default)
   - Or select specific regions

3. **Content Rating:**
   - Already done above

4. **Ads:**
   - "No ads shown" (if applicable)

5. Click **Save**

### 5. Add Testers for Closed Testing

**Location:** Testers → Closed testing

1. Click **Closed testing** (left menu)
2. Click **Create email list** (or use existing)
3. Add 12+ testers:
   - Gmail addresses
   - Test accounts
   - Friends/colleagues

4. Share test link with testers
5. Wait for feedback (14 days minimum)

### 6. Request Production Access

**Location:** Release → Production

**Requirements before requesting:**
- ✅ App must be configured
- ✅ Store listing complete
- ✅ Content rating done
- ✅ Closed testing with 12+ users for 14 days
- ✅ Privacy policy set

**How to request:**
1. Go to **Release** → **Production**
2. Click **Create release**
3. Upload same APK/AAB
4. Answer production questions:
   - "How did you test this release?"
   - "What new features are in this version?"
5. Click **Review release**
6. Google reviews (typically 24-48 hours)
